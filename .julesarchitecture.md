## 2026-06-25 Graph File Writing Extraction

Extracted `graph.json` and `backlinks.json` writing logic from `internal/generator/generator.go` and `internal/sitedata/write.go` to a dedicated `WriteGraphFiles(outputDir string, g Graph, backlinks map[string][]string) error` function in the new `internal/graph/write.go` file.

This change aligns with the refactor-one-seam routine, centralizing the handling of graph-specific data (including sorting backlinks for deterministic output) to the `internal/graph` package, utilizing the existing `jsonutil.WriteJSON` helper. The `sitedata.Write` function now only handles `meta.json`.
