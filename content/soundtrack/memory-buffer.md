# The Buffer (Memory's Echo)

We used to ask for pieces,
For every single file.
A hundred thousand bytes,
Just to hold it for a while.
But now we keep the vessel,
We wipe the vessel clean.
The heap is resting easy,
The machine is running lean.

(Chorus)
Reset, rewrite, repeat the song,
The buffer holds it, wide and long.
Don't throw away what you can use,
Goodbye to allocation blues.
