This document provides background context for Flow Music to help generate soundtracks that fit the vibe and lore of our ongoing project. You can use this as a system prompt or base context when generating tracks.

**The Project: "La Famille"**
"La Famille" (a playful nod to "The Family Jewels", referring to Jules) is a custom-built static site generator written in Go. It takes Markdown content and processes it into beautiful HTML sites using diverse layouts, Tailwind CSS, and varied DaisyUI themes.

**The Lead Developer: Jules**
The project operates under a "Jules-forward" development model. Jules is an advanced AI software engineer (represented by an octopus mascot) who acts as the primary maintainer and core contributor. Jules does the heavy lifting: designing architecture, writing code, running tests, fixing bugs, and managing the development lifecycle.

**The Soundtrack Development Cycle**
Music is deeply woven into the workflow of this project. Generating a new track is an official part of the development cycle. Whenever Jules completes a task—whether it's patching a security vulnerability, designing a new brutalist UI layout, or refactoring the core Go codebase—a new song description/prompt is added to the soundtrack folder.

When generating music for this project, the tracks should capture this unique dynamic: the technical precision of Go programming, the creative flair of web design, the ongoing narrative of an AI octopus leading an engineering project, and the celebratory feeling of shipping code. The genre will be specified in each individual prompt, but this context should inform the overarching personality and themes of the music.

## Update: 2024-06-20
