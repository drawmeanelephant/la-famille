# Soundtrack

This folder contains the track listings and engineering details for the project's soundtrack albums.

## Vision

The soundtrack embodies the process of working on the project and will consist of a few different albums:

1.  **Album 1: Nerd Core Boom Bap:** A boom bap explainer style album, leaning into nerd core.
2.  **Album 2: Touring Funk/Bluegrass:** A touring funk or bluegrass kind of style.
3.  **Album 3: Les Bijoux de Famille:** A French Chanson cabaret-style album exploring the AI collaboration and the pun origin of the project name.
4.  **Album 4: Vol 2 Lyrics:** A compilation of lyrical tracks capturing layout design, memory optimization, and more.
5.  **Album 5: Automation Symphony:** An instrumental electronic symphony documenting routine automation, system improvements, and security tools.
6.  **Routine Tasks Vol 1:** A Nerd Core rap album detailing the daily loops and mechanical maintenance of the codebase.

These albums will document the journey and hopefully be pretty good tracks!
