# Routine: Nightly Maintenance Pass

**Goal:** Perform one bounded maintenance pass that improves project cleanliness, consistency, or readiness without introducing broad new scope.

## Task Details
1. **Choose One Maintenance Theme:** Examples include stale docs cleanup, template consistency, content frontmatter normalization, output naming consistency, or small repo hygiene improvements.
2. **Limit Scope:** Finish one maintenance slice completely.
3. **Verify Stability:** Ensure the cleanup does not break generation, tests, or expected output.
4. **Leave a Clear Result:** The repo should be measurably cleaner or more consistent at the end of the pass.

## Execution Reminders
* Avoid mixing cleanup with major feature work.
* Prefer consistency and reduction of friction.
* Run tests and remove incidental debris before finishing.
* **Upon successful completion, you MUST append a short log (including date, routine name, success status, and any learnings or suggestions for improving this routine) to the "Run Log" section of `content/jules/index.md`.**
