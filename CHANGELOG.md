# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0-alpha.1] - 2026-06-18

### Added
- Initial project structure for La Famille static site generator.
- Support for YAML frontmatter (Title, Author, Date).
- Markdown to HTML conversion using Goldmark.
- HTML sanitization with Bluemonday.
- Deterministic file processing and output.
- Support for internal linking and missing page stubs.
- Multi-template system (preliminary with layout.html).
- CI pipeline for automated testing and merging.
- Project Soundtrack and Albums lore.
